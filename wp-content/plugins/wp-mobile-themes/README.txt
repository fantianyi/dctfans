Display a mobile theme when the website is viewed with a mobile/tablet browser.

== Description ==

Allows you select another themes that will be sent to mobile users and tablet users.

为手机和平板用户选择另外的主题显示网站内容，手机和平板的主题可分开设定。

**Supported Languages:**

* US English/en_US (default)
* 简体中文/zh_CN ([mg12](http://www.neoease.com/))
* Slovak/sk_SK ([Branco Radenovich](http://webhostinggeeks.com/blog/))

**Demo:**

[http://www.neoease.com/](http://www.neoease.com/)

== Installation ==

1. Unzip archive to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Appearance->Mobile Themes' to select a mobile themes.
