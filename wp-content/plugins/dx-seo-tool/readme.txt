== Description ==

powerful, versatile seo plugin. Be able to self-defined home page, article pages, page classification, page, tag page title, keywords, description tags; automatically long tail keywords recommended; automatically post pictures title, alt attributes.

强大、多功能的中文seo插件。能够自定义首页、文章页、页面、分类页、标签页、自定义post type、自定义taxonomy的title、keywords、description标签；自动长尾关键词建议；自动文章图片title、alt属性；文章内容外链、标签云自动nofollow；相关文章；站点地图、自动锚文本等功能。

详情请浏览：[http://www.daxiawp.com/dx-seo-plugin.html](http://www.daxiawp.com/dx-seo-plugin.html)

== Installation ==

可以通过以下两种方法的其中一种来安装DX-seo插件：

1. 将下载的文件解压缩，然后将`dx-seo-tool`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'dx-seo'，按照提示安装启用
